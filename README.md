## 前言

此项目为基于Java的便民医疗服务小程序，是我毕业设计的实战项目。在这个项目中，我致力于为用户提供便捷的医疗服务，通过整合线上医疗资源，使患者能够更轻松地获得医疗帮助。以下是该项目详细的介绍。

## 内容介绍

便民医疗服务小程序主要包括以下功能：在线挂号、科室查询、医生预约、健康咨询等。通过这些功能，用户可以随时随地了解医疗信息，预约医生，节省了排队等候的时间。此外，后台管理系统方便医疗机构对信息进行维护和管理。

## 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
